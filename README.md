# 📝 Simple Blog (Flask CRUD Application)

## 📌 Project Overview

This project is a **Simple Blog Web Application** developed using **Flask (Python)**.
It allows users to perform basic **CRUD operations (Create, Read, Update, Delete)** on blog posts without using any database.

---

## 🎯 Features

* ✅ Create new blog posts
* ✅ View all blog posts
* ✅ Edit existing posts
* ✅ Delete posts
* ✅ Clean UI with navigation
* ✅ Responsive and simple design

---

## 🛠️ Technologies Used

* **Python**
* **Flask**
* **HTML (Jinja2 Templates)**
* **CSS**

## 🧠 How It Works

* Blog posts are stored in a **Python list (in-memory)**
* Flask handles routing and rendering
* Jinja2 templates are used for dynamic HTML pages
* No database is used in this project
